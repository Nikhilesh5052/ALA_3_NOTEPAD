# 📱 ALA 3 - Flutter Notes App

## 📌 Project Title
ADF ALA 3 - Notes Application using Flutter

---

## 📖 Project Description
This project is a Flutter-based Notes Application that demonstrates local data storage using SharedPreferences. The app allows users to create, edit, delete, and manage notes efficiently with a modern UI.

---

## 🎯 Objectives
- To understand Flutter UI development
- To implement local data storage
- To perform CRUD operations (Create, Read, Update, Delete)
- To manage app state dynamically

---

## ⚙️ Features
- Add new notes
- Edit existing notes
- Delete notes (Swipe to delete)
- Undo delete (Snackbar)
- Recycle Bin (Restore / Delete permanently)
- Dark Mode toggle
- Display notes with date (created & modified)
- Simple and clean UI

---

## 🛠️ Technologies Used
- Flutter
- Dart
- SharedPreferences (Local Storage)
- Android Studio
- Git & GitHub

---

## 🔄 Project Workflow
1. User opens the app
2. User can add a note using the + button
3. Notes are saved locally using SharedPreferences
4. Notes are displayed on the home screen
5. User can:
   - Edit notes
   - Delete notes (moves to recycle bin)
6. Deleted notes can be:
   - Restored
   - Permanently deleted
7. Dark mode can be toggled
